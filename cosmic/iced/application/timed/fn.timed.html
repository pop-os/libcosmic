Creates an Application with an update function that also takes the Instant of each Message.

This constructor is useful to create animated applications that are pure (e.g. without relying on side-effect calls like Instant::now).

Purity is needed when you want your application to end up in the same exact state given the same history of messages. This property enables proper time traveling debugging with comet.
