pub struct NoAttributes<B>
where
    B: PathBuilder,
{ /* private fields */ }
Expand description

A convenience wrapper for PathBuilder without custom attributes.

See the PathBuilder trait.

This simply forwards to an underlying PathBuilder implementation, using no attributes.
