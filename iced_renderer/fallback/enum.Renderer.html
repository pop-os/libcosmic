iced_renderer::fallback

Enum Renderer

Source 
pub enum Renderer<A, B> {
    Primary(A),
    Secondary(B),
}
Expand description

A renderer A with a fallback strategy B.

This type can be used to easily compose existing renderers and create custom, type-safe fallback strategies.
